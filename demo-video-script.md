# Peak Overwatch — Demo Video Script
**For TikTok Developer Portal Application Submission**
**Target length:** 2-3 minutes

---

## Scene 1: Landing Page (10s)
- Show peakoverwatch.com in browser
- Scroll through the landing page briefly
- Highlight "Connect with TikTok" / dashboard preview

## Scene 2: Login Flow (15s)
- Click "Launch Dashboard" → app.peakoverwatch.com
- Click "Connect with TikTok" button
- TikTok OAuth authorization page appears
- Show the scopes being requested (user.info.basic, video.list)
- Click "Authorize"

## Scene 3: OAuth Callback (5s)
- Show redirect back to app.peakoverwatch.com
- Loading state → dashboard appears

## Scene 4: User Profile Display (15s)
- Show connected user's TikTok profile (name, avatar)
- Highlight that this data comes from user.info.basic scope
- Show account connection status

## Scene 5: Video List (20s)
- Show the user's video list pulled from video.list scope
- Scroll through videos showing:
  - Video thumbnails
  - View counts
  - Like counts
  - Post dates
- Highlight the data is real-time from TikTok's API

## Scene 6: Dashboard Overview (15s)
- Show the portfolio overview with metrics
- GMV tracking, commission calculations
- FYP health score display
- Multiple account view

## Scene 7: Wrap Up (10s)
- Show Terms of Service page (/terms)
- Show Privacy Policy page (/privacy)
- Back to dashboard — clean, professional finish

---

## Technical Notes
- Record at 1080p minimum
- Use sandbox environment (TikTok requirement for first-time apps)
- Domain in browser must match the Web URL in the application (app.peakoverwatch.com)
- All requested scopes must be visibly demonstrated
- Format: MP4 or MOV, under 50MB
